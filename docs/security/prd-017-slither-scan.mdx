## Pre-Mainnet Action Items

### Must Fix
1. **H-3** — Add `nonReentrant` modifier to `TAGITEmissions.distributeEpoch()` or move state writes before the `token.mint()` external call
2. **H-4** — Add `nonReentrant` modifier to `TAGITStaking.notifyRewardAmount()` or move state writes before `safeTransferFrom()`
3. **M-1 / M-2** — Reorder arithmetic to multiply-before-divide in `TAGITAgentReputation` and `TAGITEmissions`
4. **M-8** — Add zero-address guards to all `TAGITAccountFactory` setters

### Should Fix
5. **H-1 / H-2** — Confirm governor access control is enforced and document as accepted risk if verified
6. **M-5 / M-6 / M-7** — Explicit local variable initialization
7. **L-8 / L-9** — Mark state variables as `immutable` for gas savings (`CCIPBridgeAdapter._router`, `CCIPBridgeAdapter._wTag`, `IntegrationFactory._requiredSignatures`)

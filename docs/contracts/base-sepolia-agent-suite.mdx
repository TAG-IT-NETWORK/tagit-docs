## Deploying to Base Sepolia

To reproduce this deployment from scratch:

```bash
# 1. Set env vars
export BASE_SEPOLIA_RPC_URL=https://base-sepolia.g.alchemy.com/v2/YOUR_KEY
export BASESCAN_API_KEY=YOUR_BASESCAN_API_KEY
export PRIVATE_KEY=0x...
export TAGIT_ACCESS=0x0611FE60f6E37230bDaf04c5F2Ac2dc9012130a9

# 2. Dry-run (no broadcast)
forge script script/deploy/DeployAgentSuite.s.sol \
  --rpc-url $BASE_SEPOLIA_RPC_URL

# 3. Live deploy + verify
forge script script/deploy/DeployAgentSuite.s.sol \
  --rpc-url $BASE_SEPOLIA_RPC_URL \
  --broadcast \
  --verify
```

<Callout type="info">
The suite script deploys all three contracts in a single broadcast with automatic wiring.
`TAGIT_ACCESS` must point to a pre-deployed `TAGITAccess` controller.
</Callout>

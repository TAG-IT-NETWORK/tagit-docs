# wTAGBondingCurve

An on-chain linear bonding curve that sets the price of wTAG tokens algorithmically. As more wTAG is purchased the price rises; as tokens are sold back the price falls. ETH flows in and out of the contract's reserve, ensuring every wTAG in circulation is backed.

## Overview

`wTAGBondingCurve` implements a **linear bonding curve** for wTAG price discovery. The instantaneous spot price is:

```
price(S) = initialPrice + (slope × S / PRECISION)
```

where `S` = `curveSupply` (tokens already sold through the curve) and `PRECISION = 1e18`.

**Buy cost** for `N` tokens at supply `S`:

```
cost = P₀·N/PREC + slope·N·(2S + N) / (2·PREC²)
```

**Sell return** for `N` tokens at supply `S`:

```
return = P₀·N/PREC + slope·N·(2S − N) / (2·PREC²)
```

A protocol fee (`feeBps`, max 1,000 bps = 10%) is added on buys and deducted on sells, accruing in `accruedFees` for owner withdrawal.

## Price Discovery Lifecycle

```mermaid
sequenceDiagram
    participant Buyer
    participant Curve as wTAGBondingCurve
    participant Token as wTAG ERC-20

    Buyer->>Curve: getBuyQuote(tokenAmount)
    Curve-->>Buyer: (totalCost, fee)

    Buyer->>Curve: buy{value: totalCost}(tokenAmount, maxCost)
    Curve->>Curve: validate checks (slippage, balance)
    Curve->>Curve: curveSupply += N; reserve += base; fees += fee
    Curve->>Token: safeTransfer(buyer, tokenAmount)
    Curve-->>Buyer: refund excess ETH
    Curve-->>Buyer: emit Buy(buyer, amount, cost, fee)
```

## View Functions — Price Quotes

### `getBuyQuote`

```solidity
function getBuyQuote(uint256 tokenAmount)
    external view
    returns (uint256 totalCost, uint256 fee)
```

Returns the total ETH cost (base + fee) and fee-only portion to buy `tokenAmount` wTAG at the current supply.

### `getSellQuote`

```solidity
function getSellQuote(uint256 tokenAmount)
    external view
    returns (uint256 netReturn, uint256 fee)
```

Returns the net ETH return after fee deduction and the fee-only portion for selling `tokenAmount` wTAG.

### `currentPrice`

```solidity
function currentPrice() external view returns (uint256 price)
```

Returns the current spot price per full wTAG token (1e18 units) in wei.

## Trading Functions

### `buy`

```solidity
function buy(uint256 tokenAmount, uint256 maxCost)
    external payable
    nonReentrant whenNotPaused
```

Buys `tokenAmount` wTAG tokens with ETH. Slippage protection via `maxCost`. Excess ETH is refunded.

| Parameter | Type | Description |
|-----------|------|-------------|
| `tokenAmount` | `uint256` | Number of wTAG to buy (18-decimal units) |
| `maxCost` | `uint256` | Maximum ETH willing to pay (wei) |

Reverts: `ZeroAmount`, `SlippageExceeded`, `InsufficientPayment`, `InsufficientBalance`

### `sell`

```solidity
function sell(uint256 tokenAmount, uint256 minReturn)
    external
    nonReentrant whenNotPaused
```

Sells `tokenAmount` wTAG back to the curve for ETH. Caller must first `approve` this contract. Slippage protection via `minReturn`.

| Parameter | Type | Description |
|-----------|------|-------------|
| `tokenAmount` | `uint256` | Number of wTAG to sell (18-decimal units) |
| `minReturn` | `uint256` | Minimum ETH expected (wei) |

Reverts: `ZeroAmount`, `ExceedsCurveSupply`, `SlippageExceeded`

## Admin Functions

### `updateParams`

```solidity
function updateParams(uint256 _initialPrice, uint256 _slope, uint256 _feeBps)
    external onlyOwner
```

Updates bonding curve parameters. Applies full validation (same as constructor). Only callable by owner.

> **Note**: Parameters can only be changed while `curveSupply == 0` to protect existing participants.

### `withdrawFees`

```solidity
function withdrawFees() external onlyOwner nonReentrant
```

Transfers all `accruedFees` to the owner. Reverts `NoFeesToWithdraw` if balance is zero.

### `pause` / `unpause`

```solidity
function pause() external onlyOwner
function unpause() external onlyOwner
```

Emergency circuit-breaker. While paused, `buy()` and `sell()` revert. View functions and `withdrawFees` remain accessible.

## Security Model

| Mechanism | Applied To | Purpose |
|-----------|-----------|---------|
| `ReentrancyGuard` | `buy`, `sell`, `withdrawFees` | Prevents re-entrancy via ETH callbacks |
| `Pausable` | `buy`, `sell` | Emergency circuit-breaker |
| `Ownable` | `updateParams`, `withdrawFees`, `pause/unpause` | Admin operations |
| CEI pattern | All state-changing functions | State updates before external calls |
| Slippage guard | `buy` (`maxCost`), `sell` (`minReturn`) | Front-running / sandwich protection |

## TypeScript SDK Usage

```typescript
import { createPublicClient, createWalletClient, parseEther } from "viem";

// Get a buy quote
const [totalCost, fee] = await publicClient.readContract({
  address: BONDING_CURVE_ADDRESS,
  abi: wTAGBondingCurveAbi,
  functionName: "getBuyQuote",
  args: [parseEther("100")], // 100 wTAG
});

// Execute buy with 1% slippage tolerance
const maxCost = (totalCost * 101n) / 100n;
await walletClient.writeContract({
  address: BONDING_CURVE_ADDRESS,
  abi: wTAGBondingCurveAbi,
  functionName: "buy",
  args: [parseEther("100"), maxCost],
  value: maxCost,
});
```
